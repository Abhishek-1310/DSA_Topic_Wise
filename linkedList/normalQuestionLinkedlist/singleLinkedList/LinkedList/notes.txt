This is a very important linked list pattern.

1) Use current !== null when you want to visit every node
while(current !== null){
    console.log(current.data);
    current = current.next;
}

Example: 10 -> 20 -> 30 -> null
Visits: 10 20 30
Used for: Count nodes , Print list, Search a value, Sum values

2) Use current.next !== null when you need to stop at the last node
while(current.next !== null){
    current = current.next;
}

Example: 10 -> 20 -> 30 -> null
Execution:

current = 10
current = 20
stop at 30

Notice: after the loop, current is the last node.

Used for: Finding the last node, Appending in a head-only linked list
