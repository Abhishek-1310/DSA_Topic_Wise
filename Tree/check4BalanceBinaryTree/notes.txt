110. Balanced Binary Tree
Given a binary tree, determine if it is height-balanced.

Example 1:

Input: root = [3,9,20,null,null,15,7]
Output: true
Example 2:

Input: root = [1,2,2,3,3,null,null,4,4]
Output: false
Example 3:

Input: root = []
Output: true

sol:- balance BT => for every Node:- abs(height(left) - height(right)) <= 1;
we can say tree is balance bt:- where the height of left and right subtree of every node diff by atmost 1
