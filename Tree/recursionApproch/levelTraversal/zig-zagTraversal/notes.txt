103. Binary Tree Zigzag Level Order Traversal
Given the root of a binary tree, return the zigzag level order traversal of its nodes' values. (i.e., from left to right, then right to left for the next level and alternate between).

Example 1:
        3          ->3 
      /   \
     9     20      9 <- 20<-
          /  \
         15   7   ->15 -> 7
Input: root = [3,9,20,null,null,15,7]
Output: [[3],[20,9],[15,7]]


notes:- Method	What it does	Returns	Example
shift()  	Removes the first element	       The removed element	[1,2,3] → shift() → returns 1,   array becomes [2,3]
unshift()	Adds element(s) at the beginning	New array length	[2,3] → unshift(1) → returns 3, array becomes [1,2,3]
