how to solve with backtracking

✅ Normal Recursion (what you did)
return n + SumN(n-1);
👉 Values are returned while coming back (bottom → up)

🔥 Backtracking Style
👉 You carry state forward and update it
👉 No need to depend on return value

Recursion:
“Give me answer of smaller problem”

Backtracking:
“I’ll carry answer with me”

Backtracking is a recursive technique where we explore a path, maintain state, and then undo (backtrack)
the state when returning to try other possibilities
state is important in backtracking

function SumN(n, sum){
    if(n === 0){
        console.log(sum);
        return;
    }
    SumN(n - 1, sum + n);
}
SumN(5, 0);
