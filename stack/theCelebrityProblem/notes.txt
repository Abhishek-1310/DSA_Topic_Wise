  element of row i and column j is set to 1 it means ith person knows jth person. You need to return the index 
  of the celebrity in the party, if the celebrity does not exist, return -1.

Note: Follow 0-based indexing.
Examples:            A  B  C
Input: mat[][] = [A [1, 1, 0],
                  B [0, 1, 0],
                  C [0, 1, 1]]
a know a,b  and b know only b , c know b ,c all know b but b don't know anyone
