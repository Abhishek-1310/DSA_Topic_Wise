Min Stack
Design a stack that supports push, pop, top, and retrieving the minimum element in constant time.

Implement the MinStack class:

MinStack() initializes the stack object.
void push(int val) pushes the element val onto the stack.
void pop() removes the element on the top of the stack.
int top() gets the top element of the stack.
int getMin() retrieves the minimum element in the stack.
You must implement a solution with O(1) time complexity for each function.

explain:- When you push a new minimum, the old minimum gets overwritten (this.min = val). Later when you pop that
value, how do you know what the minimum was before it? You need to somehow store the old minimum — but you can't
use extra space.
The trick:
Instead of storing val directly, store encoded = 2 * val - this.min_before. This encoded value is always less 
than the new minimum (since val < this.min_before), which is how getMin() and pop() know the stack has an encoded 
value.

Step 1: Push 5
It's the first element. We push it normally and set min_val. Stack: [5]. min val: 5

Step 2: Push 7
Is 7 less than min_val (5)? No. * Since it's not a new minimum, we just push it normally.        min_val stays 5.
Stack: [5, 7] min_val: 5

Step 3: Push 3
Is 3 less than min_val (5)? Yes! We have a new minimum. Instead of pushing 3, we push the encoded value:

Encoded = 2 × New Val - Old Min
Encoded = 2×3-5-1
We update min val to 3.
Stack: [5, 7, 1] (Notice the top is 1, not 3!).       min_val: 3

decode 


Now our stack looks like [5, 7, 1] and min_val is 3, Let's pop elements one by one.

Pop 1: Removing the 3

We want to pop the top element. The top of the stack
The Tripwire Check: Is the top element (1) less than min_vol (3)? Yes! This tells us two things instantly:

1. The actual value the user pushed here was the current min_val (3).
2. We need to restore the old minimum before we pop this.

The Decoding Math:
Old Min = 2 × min_val - Top Value
Old Min = 2 ×3-1=5
We update min_val back to 5 and pop the 1.
Stack is now: [5, 7]

min_val is now: 5 (Successfully restored!)
