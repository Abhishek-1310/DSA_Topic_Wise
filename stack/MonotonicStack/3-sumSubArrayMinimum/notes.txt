Sum of Subarray Minimums
Given an array of integers arr, find the sum of min(b), where b ranges over every (contiguous) subarray of arr. Since
 the answer may be large, return the answer modulo 109 + 7.

Example 1:

Input: arr = [3,1,2,4]
Output: 17
Explanation: 
Subarrays are [3], [1], [2], [4], [3,1], [1,2], [2,4], [3,1,2], [1,2,4], [3,1,2,4]. 
Minimums are 3, 1, 2, 4, 1, 1, 2, 1, 1, 1.
Sum is 17.

Missing MOD (LeetCode requirement)

For large input:

sum can overflow

👉 Use:

const MOD = 1e9 + 7;
1e9 + 7 = 1000000007

👉 It’s a large prime number

🧠 Why do we use modulo at all?

In problems like yours:

sum += arr[i] * l[i] * r[i]

👉 Values can become very large
👉 Can overflow or lose precision in JavaScript
so we use sum%mod


sum += arr[i] * l[i] * r[i] % MOD;
👉 Issue:

Only arr[i] * l[i] * r[i] is moduloed
But sum keeps growing without modulo

👉 So sum can exceed JS safe integer range (Number.MAX_SAFE_INTEGER)
sum = (sum + (arr[i] * l[i] * r[i]) % MOD) % MOD;
