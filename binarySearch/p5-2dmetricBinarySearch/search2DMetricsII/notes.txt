240. Search a 2D Matrix II
Write an efficient algorithm that searches for a value target in an m x n integer matrix matrix. This matrix has the following properties:

Integers in each row are sorted in ascending from left to right.
Integers in each column are sorted in ascending from top to bottom.
 

Example 1:

Input: matrix = [[1, 4, 7, 11,15],
                 [2, 5, 8, 12,19],
                 [3, 6, 9, 16,22],
                 [10,13,14,17,24],
                 [18,21,23,26,30]], target = 5
Output: true

solution:- 
Start at: matrix[0][cols-1]

Why top-right?
Because from there:
left side is smaller down side is larger

So we can eliminate one full row or one full column each step.

If current > target
Move left => c-- =>because everything below is even larger.

If current < target
Move down => r++ => because everything left is even smaller.
